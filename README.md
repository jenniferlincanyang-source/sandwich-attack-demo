# 三明治攻击演示 (Sandwich Attack Demo)

一个交互式前端网站，详细介绍加密货币交易中的"三明治攻击"（Sandwich Attack）。

## 在线预览

访问 GitHub Pages: https://jenniferlincanyang-source.github.io/sandwich-attack-demo/

## 功能特点

- **概念介绍**: MEV、Mempool、滑点等核心概念解释
- **攻击机制**: 四步攻击流程可视化展示
- **攻击类型**: 基础三明治、多层三明治、JIT流动性、跨池攻击
- **交互模拟**: 可调节参数的实时攻击模拟器
- **MEV基础设施**: 超级节点、区块构建者、搜索者介绍
- **防护措施**: 实用的防护建议和工具推荐

## 技术栈

- HTML5
- CSS3 (响应式设计)
- Vanilla JavaScript
- Canvas API (图表绘制)

## 本地运行

直接在浏览器中打开 `index.html` 即可。
